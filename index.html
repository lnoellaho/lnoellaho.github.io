SD-AKAW

AKAW - San Diego Surf Spots

Full-stack React web application that allows users to log in, view, and check into different surf spots in the San Diego area. Decoupled frontend(React, jQuery, Bootstrap) and backend(Node, Express, Sequelize, PostgreSQL). Utilizes password encryption libraries, a RESTful API, and surf data pulled from the Spitcast public API. Developed using agile methodologies.


Technologies: React, JavaScript, Node.js, Express, Sequelize, PostgreSQL, jQuery, Bootstrap, CORS, RESTful API, Spitcast API, Heroku
